# ProofChain Template

## Sobre o desafio
Template oficial do desafio ProofChain do Hackathon Web3 RESTIC 29.

## Objetivo
Construir uma solução auditável utilizando blockchain.

## Requisitos mínimos
- Registro on-chain
- Consulta pública
- Contrato deployado em testnet
- README funcional
- Vídeo-pitch

## Tecnologias sugeridas
- Solidity
- Hardhat
- Sepolia
- Ethers.js
- React

## Estrutura
/contracts
/frontend
/scripts
/test
/docs

## Como iniciar

### Instalar dependências
```bash
npm install
```

### Compilar contratos
```bash
npx hardhat compile
```
